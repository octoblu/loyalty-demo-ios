Loyalty Demo
===========

Loyalty Demo app using MeshbluBeaconKit and MeshbluKit

# Installation:

1. Install Xcode
2. Install Cocoapods
  * `sudo gem install cocoapods`
  * `pods setup`
3. Clone this repo
  * `git clone git@github.com:octoblu/loyalty-demo-ios`
  * `cd loyalty-demo-ios/loyalty-demo`
  * `pod install`
4. Open `loyalty-demo.xcworkspace`
